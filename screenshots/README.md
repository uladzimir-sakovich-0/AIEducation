# Screenshots

This folder contains application screenshots referenced in the main README.

Please save the following screenshots with these exact filenames:

1. `login.png` - Login page
2. `register.png` - Registration page
3. `transactions.png` - Transactions list page
4. `transaction-modal.png` - New transaction modal
5. `accounts.png` - Accounts list page
6. `account-modal.png` - New account modal
7. `categories.png` - Categories list page
8. `category-modal.png` - New category modal
9. `health.png` - Health check page

## Image Guidelines

- Format: PNG (preferred) or JPG
- Recommended width: 1200-1600px
- Keep file sizes reasonable (< 500KB each)
- Use descriptive, clear screenshots showing the main features
